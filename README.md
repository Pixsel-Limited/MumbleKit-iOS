# MumbleKit

Mumble SDK for iOS

## Requirements

- iOS 6.0

## Installation

MumbleKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MumbleKit'
```
